# 🧪 Adversarial AI Lab

> **Mission:** Securing the AI models that power the future.

This repository focuses on **Adversarial Machine Learning**, specifically the defense mechanisms required to protect Generative AI and LLMs from exploitation.

## ⚙️ Why this matters

As AI agents gain the ability to call tools (APIs, Databases), a single prompt injection can lead to unauthorized code execution or data theft. This lab builds the "Air Gap" between untrusted users and privileged AI agents.
